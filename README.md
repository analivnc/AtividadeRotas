#O que é rota dinâmica?
Rota dinâmica no Angular é uma técnica que utiliza parâmetros na URL (como /produto/:id) para carregar componentes dinamicamente com base no conteúdo, em vez de criar uma rota estática para cada item. Ela permite que o Angular reconheça padrões, capturando valores variáveis e usando-os, por exemplo, para buscar dados de um usuário ou produto específico.
#O que é paramMap?
O paramMap no Angular é uma ferramenta prática usada para ler informações dinâmicas que vêm na URL (os parâmetros de rota) de uma forma simples e organizada. 
Imagine que você tem um site de produtos e a URL é ://meusite.com. O 123 é o ID do produto que muda. O paramMap é o mecanismo que te ajuda a pegar esse "123" dentro do código TypeScript
